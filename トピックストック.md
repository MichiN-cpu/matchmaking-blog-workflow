# 📌 ブログ記事の種（トピックストック）

**記事の種を書く・見る → [topics/stock.md](topics/stock.md)**

---

## 常にすぐ開けるようにするには

1. **ピン留め**  
   [topics/stock.md](topics/stock.md) を開き、タブを**右クリック** → **「ピン留め」**を選ぶと、他のファイルを開いてもタブが残り、いつでもクリックで戻れます。

2. **クイックオープン**  
   `Cmd + P`（Mac） / `Ctrl + P`（Windows）を押して「**stock**」と入力 → Enter で `topics/stock.md` が開きます。

思いついたタイトルやキーワードは、その場で [stock.md](topics/stock.md) に追記しておくと便利です。
