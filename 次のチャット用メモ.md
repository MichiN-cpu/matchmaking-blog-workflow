# 次のチャットで続けるとき用メモ

新しいチャットを始めるとき、AI にこのファイルを参照させると、長い会話履歴なしで続きから進められます。

---

## このプロジェクトでやっていること

- **結婚相談所ブログ**の記事の種を `topics/stock.md` にストック
- 毎日1トピックで記事下書きを作成 → あなたが内容を確認 → OK なら画像・Wix用・GBP用を `drafts/` に保存
- 品質チェックは `EDITORIAL_GUIDELINES.md`（顧客目線・失敗点・多角的観点など）
- Claude Code で Gemini 等に指示を回す運用は `SETUP.md`、日次手順は `WORKFLOW.md`

## 新しいチャットでよく使う依頼例

- 「今日のブログ書いて。topics/stock.md から1トピック選んで、EDITORIAL_GUIDELINES に沿って。出したら止まって確認させて」
- 「OK。画像生成して、drafts に保存、used/log.md に記録して」
- トピックを足したいときは `topics/stock.md` を直接編集

## 参照してほしいファイル

- `README.md` … 全体像
- `WORKFLOW.md` … 日次の流れ・依頼文
- `EDITORIAL_GUIDELINES.md` … 記事の品質・観点
- `SETUP.md` … Claude Code・マルチAI

---

**使い方**: 新チャットで「matchmaking-blog-workflow の 次のチャット用メモ.md を読んで」と送ると、この内容を踏まえて対応できます。
