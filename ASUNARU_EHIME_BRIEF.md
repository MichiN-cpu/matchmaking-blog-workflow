# あすなる愛媛 ブログ・記事の共通方針（AI用ブリーフ）

**このファイルは Cursor / Claude Code / ChatGPT など、ブログ記事作成に携わるAIが共通で参照するためのものです。**  
あすなる愛媛の結婚相談所ブログで「輝く人たちの入会」を増やし、「諦めないで柔軟に取り組む会員」を育て、成婚退会数を増やすことを目指します。

---

## 1. あすなる愛媛の立ち位置（伝え方の前提）

- **あすなる愛媛**は結婚相談所（愛媛）。運営3年目、ご成婚退会は現時点で4組。
- **歴史の長さや成婚退会の「数」は強調しない。** 実績の多さで他社と比較されると不利になるため、記事内でも「実績数・歴史を選びの軸にしない」というメッセージを自然に含めつつ、**別の選び軸**を前面に出す。

---

## 2. 記事で伝えたい「あすなる愛媛の良さ」

以下のような強みが伝わるように記事を書く（押し売りにならない範囲で）。

| 伝えたいこと | 記事での表し方の例 |
|--------------|----------------------|
| **一人ひとりに寄り添う** | 少人数・手厚いサポート、その人に合った進め方、相談しやすい雰囲気 |
| **諦めないで柔軟に取り組む** | 型にはめない、ペースに合わせる、うまくいかないときも一緒に考える、次の一手を提案 |
| **「数字」より「どんなサポートか」** | 実績の数より、面談の仕方・プロフィール添削・お見合いの進め方・継続的なフォローの中身 |
| **愛媛で婚活する人を応援** | 地域に根ざした相談、オンラインも活用した柔軟な進め方（忙しくても・場所が遠くても） |
| **会員が輝く・自分らしく進められる** | 押しつけではなく、その人らしい婚活の形を一緒に探す、安心して話せる場 |

- 既存ブログの特徴・トーンは、別途 Claude Code や ChatGPT に伝えている前提。このブリーフは「**何を強調し、何を強調しないか**」の共通ルールとして使う。

---

## 3. 記事で避けたい表現・トーン

- **歴史・創業年・〇〇年目の実績**を前面に出す表現。
- **成婚退会数・成婚率・会員数**などの「大きな数字」を売りにする表現。
- 「実績が豊富だから選ぶ」という選び方を促す表現。（代わりに「サポートの中身・相性・続けやすさ」で選ぶ流れにする。）

---

## 4. 記事のゴール（共通）

- **見込み会員**：あすなる愛媛のような「寄り添い型・少人数で手厚い」相談所の良さが伝わり、無料相談に興味を持ってもらう。
- **会員**：諦めず、柔軟に取り組む姿勢が育つような記事（型・ノウハウ・心構えを伝えつつ、押しつけない）。
- 結果として、**入会数の増加**と**成婚退会数の増加**に繋がることを目指す。
